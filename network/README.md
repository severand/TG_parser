# Network Layer - HTTP Requests

Networking operations with anti-detection features.

## Modules

- **http_client.py** - Main HTTP client with timeout and retry logic
- **session_handler.py** - Session management
- **delay_generator.py** - Random delay generation for anti-detection
- **headers_rotator.py** - User-Agent and header rotation
- **cookie_manager.py** - Cookie persistence
- **user_agents.py** - List of User-Agents

## Features

- Automatic User-Agent rotation
- Header randomization
- Delay between requests
- Cookie management
- Retry mechanism
- Timeout handling
